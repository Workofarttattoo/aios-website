## Part 2: REST API Endpoints

### API Overview

**Base URL:** `https://api.aios.is/v1`
**Authentication:** Bearer token (JWT)
**Format:** JSON
**Rate Limit:** 1,000 requests/hour per API key
**Status Codes:** Standard HTTP (200, 201, 400, 401, 403, 404, 429, 500)

### Core Endpoints
